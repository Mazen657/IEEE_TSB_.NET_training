## 📁 Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [Inventory Management System](./Inventory-Management-System) | Console app to add, view, and update products — stores up to 50 products with Name, Quantity, and Price |
| 2 | [Quiz Game](./Quiz-Game-Project) | Console quiz game with 3 questions, answer validation, and final score tracking |
| 3 | [Random Values Generator](./Random-Values-Generator) | Generates random numbers in a range or random strings using capital letters, small letters, symbols, and numbers |
| 4 | [Task Tracker](./Task-Tracker) | Console app to add, view, mark as completed, and delete tasks — supports up to 100 tasks |

## 🚀 How to Run Any Project

```bash
# 1. Clone the repo
git clone https://github.com/your-username/repo-name.git

# 2. Navigate to the project folder
cd project-name

# 3. Run the project
dotnet run
```

> ⚠️ Make sure you have **.NET SDK** installed on your machine.
